# AI Detector - AI vs Human Text Classification

A machine learning system that detects whether text is written by humans or generated by AI. This project uses a variety of linguistic and statistical features to classify text with high accuracy.

## 🎯 Project Overview

This project implements a text classification system that distinguishes between human-written and AI-generated text. It leverages multiple feature extraction techniques including stylometry analysis, perplexity measurement, and paraphrase detection to build a robust predictive model.

## 📋 Features

- **Multi-Feature Extraction**: Combines multiple text analysis techniques:
  - Stylometry analysis
  - Perplexity calculation
  - Paraphrase detection
  - Text preprocessing and normalization
  
- **Machine Learning Classification**: Uses Random Forest classifier for robust predictions
- **Confidence Scoring**: Returns probability scores along with predictions
- **Easy Inference**: Simple command-line interface for making predictions on new text

## 🚀 Quick Start

### Prerequisites

- Python 3.7+
- pip package manager

### Installation

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Download spaCy language model (required for text processing):
```bash
python -m spacy download en_core_web_sm
```

### Usage

#### Interactive Prediction

Run the main script to interactively predict text:

```bash
python main.py
```

Enter your text when prompted, and the system will output whether it's AI or human-generated with confidence score.

#### Train a New Model

If you have new training data in `data/ai_human_detection_v1.csv`:

1. Build the feature dataset:
```bash
python build_dataset.py
```

2. Train the model:
```bash
python -m src.train_model
```

#### Test Inference

Run the test inference script:
```bash
python test_inference.py
```

## 🔍 Feature Extraction

The system extracts the following features from text:

1. **Stylometric Features**:
   - Average word length
   - Sentence complexity
   - Lexical diversity
   - Punctuation patterns

2. **Perplexity-based Features**:
   - Text perplexity scores
   - Language model confidence

3. **Paraphrase Features**:
   - Paraphrase likelihood
   - Semantic coherence

4. **Text Properties**:
   - Word count
   - Character count
   - Unique word ratio

## 🤖 Model Architecture

- **Algorithm**: Random Forest Classifier
- **Number of Trees**: 100
- **Input Features**: 30+ extracted linguistic features
- **Output**: Binary classification (AI / Human)
- **Confidence**: Probability scores for both classes

## 📊 Data Format

Training data should be in CSV format with the following columns:

| Column | Type | Description |
|--------|------|-------------|
| `text` | string | The text content to analyze |
| `human_or_ai` | string/int | Label: "human", "ai", or equivalent (0/1) |

## 🔧 Configuration

Feature extraction and model parameters can be adjusted in:
- `src/feature_pipeline.py`: Adjust feature extraction
- `src/train_model.py`: Modify model hyperparameters
